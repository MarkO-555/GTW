Retrochallenge 2016 Warm-Up.


Instructions:

Object: annihilate all other players.

UP key: menu cursor up / power up
DOWN key: menu cursor down / power up
LEFT key: angle left
RIGHT key: angle right
SPACE key: menu select / fire missile
X key: rotate displays:
    menu/scores -> debug log -> playfield
Y key: leaves immediately/quits (in playfield only)
Z key: blows yourself up. (just for testing the nuclear animations...)


Only one server may operate in the IRC channel, and it should be the first to join the channel. The channel is:

#coco_war irc.freenode.net port 6667

Tested on:
  Real CoCo3
  Vcc CoCo3
  Mess CoCo3
  Mess CoCo2

at setup it asks you "'becker' port?"  press Y if you're using an emulator, and N if you wish to use the regular bitbanger DW for real cocos.
